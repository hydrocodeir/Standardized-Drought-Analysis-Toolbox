Standardized Drought Analysis Toolbox (SDAT)

A client-side dashboard for deriving nonparametric standardized drought indices from station and gridded datasets.

The Standardized Drought Analysis Toolbox offers a generalized framework for deriving statistically comparable drought indicators without forcing a parametric distribution fit. It can be applied to precipitation, soil moisture, relative humidity, runoff, streamflow, groundwater level, surface water supply, and water storage data.

Welcome to SDAT
Standardized Drought Analysis Dashboard

What This Dashboard Does

SDAT derives nonparametric standardized drought indicators from station CSV/TXT and gridded NetCDF datasets, then visualizes the results as time series and map layers.

Where It Comes From

The workflow follows the SDAT MATLAB toolbox and the published nonparametric standardized drought framework by Farahmand and AghaKouchak, with project history added for dashboard use.

How To Start

Create a project, upload station or NetCDF data, select monthly scales, run the browser-side calculation, and move through stations, scales, and map dates interactively.
